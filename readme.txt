== Description ==

Ce widget vous permet d'afficher votre profil BDManiac.fr et vos dernières lectures sur votre site.

== Installation ==

1. Uploadez le fichier `bdmaniac-widget.php` dans le répertoire `/wp-content/plugins/` de votre ftp
2. Activez le plugin dans l'onglet 'Plugins' votre panel admin wordpress
3. Allez dans l'onglet 'Widgets' de votre admin, placez le où vous désirez et entrez votre nom d'utilisateur BD Maniac
